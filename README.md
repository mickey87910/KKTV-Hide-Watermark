## Usage  
### Chrome應用商店下載（推薦）
<a href="https://chrome.google.com/webstore/detail/kktv-hide-watermark/bhdmolgfhgembgjibflpfmgpbgmbmpgo" target="_blank">前往Chrome應用商店</a>  
### Github安裝 (請按照下列步驟安裝)
#### 1.下載最新釋出的版本  
<a href="https://github.com/omega87910/KKTV-Hide-Watermark/releases" target="_blank">GitHub下載區</a>  
#### 2.解壓縮下載好的zip檔案  
解壓縮完畢的資料夾長這樣  
<img src="https://github.com/omega87910/MY_GITHUB_IMAGES/blob/master/KKTV-Hide-Watermark/list.PNG" width="40%">  
#### 3.新增解壓縮好的資料夾  
請先於網址列輸入``chrome://extensions/``前往Chrome的套件管理中心  
接著點擊左上角``載入未封裝項目``，並選取剛剛的資料夾  
<img src="https://github.com/omega87910/MY_GITHUB_IMAGES/blob/master/KKTV-Hide-Watermark/import.PNG" width="70%">
